The first public preview build of **SmartGit 26.1** is available for download.

The most important new features/improvements are:

- AI features: "Ask AI" dialog for custom prompts, Edit Message AI suggestions, improved commit annotation
- Worktrees: state indication (clean/modified), drag & drop support for clean worktrees
- GUI overhaul: reworked toolbar icons with dark theme variants, improved Compare colors, darker dark theme
- Move & Squash: support for "fixup!" and "squash!" commit message prefixes
- LFS Lock: can be performed on multiple files at once
- Standard window: added Submodule commands (Add, Initialize, Synchronize, Reset, Deactivate, Deinit, Unregister)
