We are delighted to announce the official release of [SmartGit 24.1](https://www.syntevo.com/smartgit/).
First, we want to thank all users who have tested the preview versions and provided valuable feedback.
Without you, the progress would not have been possible!

SmartGit 24.1 comes with

- many improvements for the **Standard window**:
	- easier submodule switching
	- fixing case-changed files
	- subdirectory log
	- open at hosting provider
- largely improved **cloning from hosting providers**,
- selecting **GitHub issues** (as commit message),
- support for **multiple accounts** of same hosting provider,
- the file log can **follow copies**,
- in the Changes view **irrelevant whitespace** can be hidden,
- the new **Rollback** (power-user) command has been added,
- SmartGit can be used as **credential helper** for Git command line,
- and many more smaller improvements.

Check out [What's New](https://www.syntevo.com/smartgit/whats-new) for a more detailed list of improvements in version 24.1.

SmartGit is a graphical front-end for the distributed version control systems Git which also can be used to work with Subversion repositories.
SmartGit runs on 64-bit systems of Windows 10-11, macOS 10.14-14 and Linux (GTK 3.24+).
