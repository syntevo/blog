We are proud to announce the official release of [SmartGit 2.1](http://www.syntevo.com/smartgit/). SmartGit is a graphical front-end for the distributed version control system Git and runs on Windows, Mac OS X and Linux.

SmartGit 2.1 release focuses on a new kind of SVN integration, which supports almost the complete feature set of Subversion. For details refer to the [SmartGit SVN client introduction](http://www.syntevo.com/git-svn/index.html).
